# Eco-Marketplace: A Sustainable Shopping Guide

## Project Description:

Welcome to Eco-Marketplace, a web application designed to help users find and buy eco-friendly products. This platform acts as a guide, aggregating information on sustainable goods from various online sources and providing a single, searchable database. Our mission is to make conscious consumption easier by providing users with a comprehensive resource for eco-friendly shopping, filtered by product type and location.

## Scope
- EcoMarket is a revolutionary platform that aggregates eco-friendly products from multiple sustainable marketplaces across the web.
- Web application designed to help users find and buy eco-friendly products. 
- We believe in making sustainable shopping accessible, convenient, and transparent.
- This platform acts as a guide, aggregating information on sustainable goods from various online sources and providing a single, searchable database.

## Challenges
- Understanding Market Requirements:
    Clarified: sustainability, eco-friendly product categorization
    Decided: personal care category first
- Data Collection Strategy:
    Picked 4 sites as initial scope
    Created strategy for incremental scraping
- Scraping Multiple Unstructured Websites:
    Each website had different HTML structure.
    Dynamic content / missing selectors.
    Pagination inconsistency.
    Had to standardise extracted fields: title, image, price, brand, seller.
- Project Structuring & Planning:
    Broke down the project into modules:
    Data Scraping
    Data Cleaning
    Data classification
    User features
    Background jobs/tasks to scrape useful data periodically
